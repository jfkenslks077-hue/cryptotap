📋 Инструкция
1. Зайди на console.firebase.google.com
2. Создай проект (Add project) → без Google Analytics → Create
3. Слева: Build → Realtime Database → Create Database
4. Выбери регион (europe-west1) → Start in test mode → Enable
5. Иконка шестерёнки → Project settings → вкладка General
6. Прокрути вниз → Your apps → нажми </> (Web) → Register
7. Скопируй весь firebaseConfig → вставь ниже
